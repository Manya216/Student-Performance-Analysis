📊 Student Performance Analysis using MySQL

This project focuses on analyzing student academic performance using structured SQL queries in MySQL. I designed a relational database with three tables — students, subjects, and marks — and imported real dataset files using CSV. The core objective was to extract meaningful insights such as total and average marks per student, subject-wise performance analysis, ranking of students using window functions, identification of top performers, and subject difficulty analysis based on averages.

All queries were written manually to demonstrate a strong understanding of joins, aggregation, grouping, ranking functions, and conditional logic using CASE statements. This project reflects my practical understanding of relational database design and analytical thinking using SQL.
